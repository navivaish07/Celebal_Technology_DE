# Celebal Technology Data Engineering Internship

This repository contains assignments and tasks completed during the CEI (Celebal Excellence Intern) Internship at Celebal Technologies.

## Program Duration

15 June 2026 – 15 August 2026

## Program Overview

The CEI Internship Program is structured to provide a comprehensive and application-oriented learning experience through:

* Guided Learning: Industry-aligned curriculum to build strong technical foundations.
* Project Experience: Hands-on work on real-world Data Engineering use cases.
* Evaluation & Certification: Performance-based assessment upon completion.

## Learning Experience

The program provides exposure to:

* Industry-relevant tools
* Real-world datasets
* Mentorship from experienced professionals

## Repository Contents

This repository contains internship assignments, projects, notebooks, and related resources completed during the program.

* Hands-on project experience
* Practical learning through guided assignments
